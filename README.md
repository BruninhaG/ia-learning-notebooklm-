# Aceleração de Aprendizagem com IA: NotebookLM e Finanças

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como um desafio de aprendizagem ativa, utilizando o **Google NotebookLM** como ferramenta central. O objetivo foi criar um ecossistema de estudo sobre finanças básicas, transformando fontes brutas em conhecimento estruturado com o auxílio de IA.

## 🛠️ Ferramentas Utilizadas
* **NotebookLM:** Para análise de fontes e geração de insights baseados em documentos.
* **Fontes:** Curadoria de 3 a 5 documentos (PDFs/Textos) sobre Educação Financeira.
* **Engenharia de Prompt:** Criação de comandos estratégicos para extrair resumos e glossários.

## 🚀 O que foi desenvolvido
1. **Curadoria de Fontes:** Seleção de materiais abertos sobre conceitos financeiros introdutórios.
2. **Caderno Temático:** Organização dos materiais dentro do NotebookLM.
3. **Guia de Estudos:**
   - **Resumos Estruturados:** Síntese dos pontos principais das fontes.
   - **Glossário:** Termos fundamentais (Selic, CDI, Inflação, etc.) definidos pela IA com base nas fontes.
   - **Prompt Library:** Conjunto de prompts testados e validados para revisões futuras.

## 💡 Insights de Aprendizagem
O uso do NotebookLM permitiu uma conexão mais profunda entre as fontes, permitindo identificar contradições e complementariedades entre os autores, exercitando o pensamento crítico durante o processo.
